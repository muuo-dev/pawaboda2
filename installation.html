Installation Guide

Follow these steps to set up PawaBoda:

  1. Clone the repository: git clone https://github.com/your-repo/pawaboda.git
  2. Navigate to the project folder: cd pawaboda
  3. Install dependencies: npm install
  4. Start the development server: npm start
  5. Run the app on an emulator or device.
